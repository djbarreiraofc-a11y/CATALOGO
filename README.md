# Catálogo Bianca Moura

Site profissional para designer de cílios Bianca Moura.

## Funcionalidades

- Catálogo completo de procedimentos
- Galeria com fotos dos trabalhos
- Formulário de agendamento via WhatsApp
- Design responsivo e moderno
- Animações suaves e efeitos visuais

## Como Fazer Deploy

### Netlify

1. Conecte seu repositório GitHub ao Netlify
2. O Netlify detectará automaticamente as configurações do `netlify.toml`
3. O build será executado automaticamente

### GitHub Pages

1. Faça push do código para o GitHub
2. Configure o GitHub Pages nas configurações do repositório
3. Escolha a branch `main` e a pasta `dist` (após fazer build)

## Desenvolvimento Local

```bash
npm install
npm run dev
```

## Build de Produção

```bash
npm run build
```
